Calmly

Calmly is a web app that helps users sleep using nature sounds. Users can choose from select sounds that play with a corresponding image or video, as well as the ability to toggle a dark mode for better user experience. On window load, if the web app is being viewed on mobile, it will deliver images for the corresponding sounds, otherwise it will display video.
