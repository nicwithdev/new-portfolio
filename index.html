Watchlist

Watchlist is a web app that allows registered users to search for movies and add them to their own custom playlists via the OMBD API. Users can then utilize full CRUD capability to edit their playlist names and movies, or delete the playlist. This project is hosted on Heroku.

Javascript, Node.js, Express.js, OMDB API, MongoDB, Mongoose, Passport.js, Google OAuth

Calmly

Calmly is a web app that helps users sleep using nature sounds. Users can choose from select sounds that play with a corresponding image or video, as well as the ability to toggle a dark mode for better user experience. On window load, if the web app is being viewed on mobile, it will deliver images for the corresponding sounds, otherwise it will display video.

React, Sass
